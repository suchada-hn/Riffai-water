# 🗺️ MAP - Data Layers Complete

## ✅ ข้อมูลที่มีในแผนที่

### 1. 🏞️ ลุ่มน้ำ (Basins)

**ชั้นมองเห็น (national, static — visual only)**  
- **แหล่งที่มา:** ขอบเขตลุ่มน้ำหลักตามกฎหมาย สทนช. (Shapefile ONWR *MainBasin_ONWR_Law*; แปลงเป็น GeoJSON แล้วจัดระเบียบพิกัด WGS84)  
- **ไฟล์ในแอป:** [`frontend/public/geojson/thailand_22_basins.geojson`](frontend/public/geojson/thailand_22_basins.geojson) — โหลดในเบราว์เซอร์ที่ `/geojson/thailand_22_basins.geojson`  
- **จำนวน polygon:** 28 ฟีเจอร์ (22 ลุ่มน้ำหลัก + เกาะ/แยกย่อยตามชุดข้อมูลเดิม)  
- **คุณสมบัติใน GeoJSON:** `id`, `mb_code`, `name` / `name_th` / `name_en`, `area_sqkm`  
- **หมายเหตุสิทธิ์ใช้:** ตรวจสอบเงื่อนไขการใช้ซ้ำจาก ONWR Open Data ก่อนใช้งาน production หากยังไม่มี license ชัดเจน

**ลุ่มน้ำนำร่องจาก API (logic ในแอป)**  
- **จำนวน:** 3 ลุ่มน้ำหลัก (`GET /api/map/basins`) — ใช้เลือกลุ่มน้ำ, sub-basin, SAR, กรองสถานี  
- **ข้อมูล:** ชื่อลุ่มน้ำ, จังหวัดที่ครอบคลุม, พื้นที่ (ตร.กม.), แม่น้ำหลัก, จำนวนสถานี (ตาม backend)  
- **การแสดงผล:** ทับบนชั้น national — polygon เน้นเมื่อเลือกลุ่มน้ำนำร่อง; เส้นประเมื่อไม่ได้เลือก

### 2. 🌊 แม่น้ำสายหลัก (Rivers)
- **จำนวน:** 7 สายน้ำสำคัญ
  1. แม่น้ำโขง (Mekong) - 4,350 กม.
  2. แม่น้ำเจ้าพระยา (Chao Phraya) - 372 กม.
  3. แม่น้ำชี (Chi) - 765 กม.
  4. แม่น้ำมูล (Mun) - 673 กม.
  5. แม่น้ำบางปะกง (Bang Pakong) - 230 กม.
  6. แม่น้ำตาปี (Tapi) - 230 กม.
  7. แม่น้ำปัตตานี (Pattani) - 214 กม.

- **ข้อมูล:**
  - ชื่อไทย/อังกฤษ
  - ความยาว (กม.)
  - ลุ่มน้ำที่สังกัด
  - แม่น้ำสาขา
  - พิกัดเส้นทาง

- **การแสดงผล:** เส้นสีน้ำเงิน หนา 3px

### 3. 🏗️ เขื่อนและอ่างเก็บน้ำ (Dams & Reservoirs)
- **จำนวน:** 8 เขื่อนสำคัญ
  1. เขื่อนภูมิพล - 13,462 ล้าน ลบ.ม.
  2. เขื่อนสิริกิติ์ - 9,510 ล้าน ลบ.ม.
  3. เขื่อนจุฬาภรณ์ - 262 ล้าน ลบ.ม.
  4. เขื่อนอุบลรัตน์ - 2,431 ล้าน ลบ.ม.
  5. เขื่อนสิรินธร - 1,966 ล้าน ลบ.ม.
  6. เขื่อนป่าสักชลสิทธิ์ - 960 ล้าน ลบ.ม.
  7. เขื่อนเขาแหลม - 5,930 ล้าน ลบ.ม.
  8. เขื่อนรัชชประภา - 5,225 ล้าน ลบ.ม.

- **ข้อมูล:**
  - ชื่อไทย/อังกฤษ
  - แม่น้ำที่สร้าง
  - ความจุ (ล้าน ลบ.ม.)
  - ความสูง (ม.)
  - จังหวัด
  - ปีที่สร้าง
  - ประเภท (อเนกประสงค์/ไฟฟ้า/ชลประทาน)

- **การแสดงผล:** ไอคอน 🏗️

### 4. 💧 สถานีวัดระดับน้ำ (Water Level Stations)
- **ข้อมูล:**
  - ชื่อสถานี
  - ระดับน้ำปัจจุบัน (ม.)
  - จังหวัด
  - ลุ่มน้ำ
  - เวลาที่วัด
  - สถานะความเสี่ยง
  - แนวโน้ม (เพิ่ม/ลด/คงที่)

- **สถานะความเสี่ยง:**
  - 🟢 ปกติ (< 3.0 ม.)
  - 🟡 เฝ้าระวัง (3.0-4.0 ม.)
  - 🟠 เตือนภัย (4.0-4.5 ม.)
  - 🔴 วิกฤต (> 4.5 ม.)

- **การแสดงผล:** จุดสีตามระดับความเสี่ยง

### 5. ☔ สถานีวัดฝน (Rainfall Stations)
- **ข้อมูล:**
  - ชื่อสถานี
  - ปริมาณฝน (มม.)
  - จังหวัด
  - เวลาที่วัด
  - ความเข้มข้น

- **ระดับความเข้มข้น:**
  - ไม่มีฝน (< 10 มม.)
  - ฝนเบา (10-35 มม.)
  - ฝนปานกลาง (35-90 มม.)
  - ฝนหนัก (> 90 มม.)

### 6. 🛰️ ข้อมูลดาวเทียม (Satellite Coverage)
- **Sentinel-2** (Optical)
  - NDVI, NDWI, MNDWI, LSWI, NDBI
  - ความละเอียด 10m
  - Cloud coverage
  
- **Sentinel-1** (SAR)
  - VV/VH polarization
  - Change detection
  - All-weather

### 7. 🌊 ความลึกน้ำท่วม (Flood Depth)
- **การคาดการณ์:**
  - 0-0.5 ม. (สีฟ้าอ่อน)
  - 0.5-1.0 ม. (สีฟ้า)
  - 1.0-1.5 ม. (สีฟ้าเข้ม)
  - 1.5-2.0 ม. (สีน้ำเงิน)
  - 2.0-2.5 ม. (สีน้ำเงินเข้ม)
  - > 2.5 ม. (สีกรมท่า)

- **การแสดงผล:** วงกลมโปร่งแสง แสดงพื้นที่คาดการณ์

## 🎯 การใช้งาน

### เปิด/ปิด Layer
1. ใช้ Sidebar ด้านซ้าย
2. เลือก checkbox ของ layer ที่ต้องการ
3. แผนที่จะอัพเดททันที

### ดูรายละเอียด
1. **คลิกที่ลุ่มน้ำ** - ดูข้อมูลลุ่มน้ำ
2. **คลิกที่แม่น้ำ** - ดูข้อมูลแม่น้ำ
3. **คลิกที่เขื่อน** - ดูข้อมูลเขื่อน
4. **คลิกที่สถานี** - ดูข้อมูลการวัด

### เลือกลุ่มน้ำ
1. ใช้ Dropdown "Select Basin"
2. เลือกลุ่มน้ำที่ต้องการ
3. แผนที่จะ zoom ไปที่ลุ่มน้ำนั้น
4. แสดงเฉพาะข้อมูลในลุ่มน้ำที่เลือก

### Refresh ข้อมูล
1. คลิกปุ่ม "Refresh Data"
2. ระบบจะดึงข้อมูลล่าสุด
3. แสดงเวลาอัพเดทล่าสุด

## 🔧 API Endpoints

```typescript
// ลุ่มน้ำ
GET /api/map/basins

// แม่น้ำ
GET /api/map/rivers

// เขื่อน
GET /api/map/dams

// สถานีวัดน้ำ
GET /api/map/water-level-map?basin_id=mekong_north

// สถานีวัดฝน
GET /api/map/stations?basin_id=mekong_north&station_type=rainfall
```

## 💡 Tips

1. **ดูภาพรวม** - ปิด layer ที่ไม่จำเป็น
2. **ดูรายละเอียด** - เลือกลุ่มน้ำเฉพาะ
3. **ติดตามสถานการณ์** - เปิด water levels + rainfall
4. **วางแผน** - เปิด dams + rivers
5. **คาดการณ์** - เปิด flood depth
